## ビルド手順

- Python 3.9 / 3.10 をセットアップしてください。パスをデフォルト設定から変更してインストールする場合は、後述するslnファイルのパス設定も変更してください。 https://www.python.org/downloads/
- サブモジュールも含めて、リポジトリをクローンしてください。
- slnファイルをVisual Studioで開き、ビルドしてください。
